ESCENARIO 1

LA SIMULACION SE HARA EN PROTEUS, SE USA EL SIMULINO UNO, UN LED AMARILLO Y SIMULADOR DE PUERTOS SE HARA CON EL PROGRAMA VIRTUAL SERIAL

PARA COMPILAR POR PRIMERA VEZ:
-HACER EL MODELO EN PROTEUS SEGUN EL ESQUEMA. PARA LA LECTURA DE LA SEÑAL DIGITAL 13 PARA EL LED AMARILLO, PARA 
 LA COMUNICACION SE USARA EL COMPIM TX 1 CON TXD 3, RX 0 CON RXD 2
-HACER EL PROGRAMA EN ARDUINO. AL TERMINAR EL PROGRAMA IR A LA OPCION EXPORTAR BINARIO COMPILADO.
-EN PROTEUS HACER DOBLE CLICK EN SIMULINO UNO, EN ELLA EN LA PESTAÑA PROGRAM FILE BUSCAR EL ARCHIVO BINARIO CREADO ANTERIORMENTE
-HACER DOBLE CLICK EN COMPIN, EN ELLA PONER EL PUERTO COM 2 PARA LA SALIDA DEL ARDUINO, CAMBIAR BAUD RATE A 9600.
-HACER EL PROGRAMA EN PYTHON, EN ESTE PROGRAMA USAR EL PUERTO COM 1, PARA LA COMUNICACION.

PARA ARRANCAR EL MODELO SE HACE:
-ARRANCAR PRIMERO EL MODELO DEL PROTEUS
-CORRER EL PROGRAMA DE PYTHON LED_1.py
-PARA PRENDER EN PAYTHON PONE 'e' Y PARA APAGAR SE USA 'a' EN EL PROTEUS, Y SE VE QUE EN PYTHON VA CAMBIANDO EL LED DEL PROTEUS

ESCENARIO 2

SE HACE LA CORRIDA DIRECTAMENTE EN ARDUINO UNO y PYTHON USANDO PUERTO COM (CABLE)

PARA COMPILAR POR PRIMERA VEZ:
-HACER EL PROGRAMA EN ARDUINO. AL TERMINAR EL PROGRAMA IR A LA OPCION HERRAMIENTA->PUERTO: COM 6. TAMBIEN IR A HERRAMIENTA->PLACA: "arduino uno"
-COMPILAR EL PROGRAMA ARDUINO Y CARGAR A LA PLACA ARDUINO UNO.
-HACER EL PROGRAMA EN PYTHON, EN ESTE PROGRAMA USAR EL PUERTO COM 6, PARA LA COMUNICACION.

PARA ARRANCAR EL MODELO SE HACE:
-ARRANCAR PRIMERO EL MODELO DEL PROTEUS
-CORRER EL PROGRAMA DE PYTHON LED_1.py
-PARA PRENDER EN PAYTHON PONE 'e' Y PARA APAGAR SE USA 'a' EN EL PROTEUS, Y SE VE QUE EN PYTHON VA CAMBIANDO EL LED DEL PROTEUS
