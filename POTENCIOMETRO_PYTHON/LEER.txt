

LA SIMULACION SE HARA EN PROTEUS, SE USA EL SIMULINO UNO, UN POTENCIOMETRO Y SIMULADOR DE PUERTOS SE HARA CON EL PROGRAMA VIRTUAL SERIAL

PARA COMPILAR POR PRIMERA VEZ:
-HACER EL MODELO EN PROTEUS SEGUN EL ESQUEMA. PARA LA LECTURA DE LA SEÑAL ANALOGICA DEL POTENCIOMETRO SE USA A0, PARA 
 LA COMUNICACION SE USARA EL COMPIM PWM 1 CON TXD 3, PWM 0 CON RXD 2
-HACER EL PROGRAMA EN ARDUINO. AL TERMINAR EL PROGRAMA IR A LA OPCION EXPORTAR BINARIO COMPILADO.
-EN PROTEUS HACER DOBLE CLICK EN SIMULINO UNO, EN ELLA EN LA PESTAÑA PROGRAM FILE BUSCAR EL ARCHIVO BINARIO CREADO ANTERIORMENTE
-HACER DOBLE CLICK EN COMPIN, EN ELLA PONER EL PUERTO COM 2 PARA LA SALIDA DEL ARDUINO, CAMBIAR BAUD RATE A 9600.
-HACER EL PROGRAMA EN PYTHON, EN ESTE PROGRAMA USAR EL PUERTO COM 1, PARA LA COMUNICACION.


PARA ARRANCAR EL MODELO SE HACE:
-ARRANCAR PRIMERO EL MODELO DEL PROTEUS
-CORRER EL PROGRAMA DE PYTHON LECTURA.py
-MOVER EL POTENCIOMETRO EN EL PROTEUS, Y SE VE QUE EN PYTHON VA CAMBIANDO EL VALOR DE LECTURA Q ENVIA EL PROTEUS
